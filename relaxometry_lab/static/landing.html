Workflow

Four steps from data to result

No scripts. No command line. Works with DICOM, NIfTI, and raw Bruker exports.

1
Load Data
Upload DICOM, NIfTI, or a Bruker study ZIP. Optionally add a segmentation mask.
2
Preview
Browse your scan in 3-plane orthographic view, scroll slices, zoom, and verify alignment.
3
Fit
Run voxel-wise curve fitting. Explore individual decay curves and ROI statistics.
4
Export
Download NIfTI maps, CSV statistics, a multi-page PDF report, or raw NumPy arrays.
Data Formats

Supported inputs

Bring your data in any of these formats — no preprocessing needed.

📁
DICOM
Single enhanced multi-frame DICOM or a stack of per-slice single-frame files. Echo time read from standard tags.
.dcm
🧊
NIfTI
One file per echo (3D) or a single 4D volume. Echo times read from BIDS sidecar JSON when present.
.nii .nii.gz
🗜️
Bruker Study ZIP
Zip a Bruker ParaVision study folder and upload it directly. Browse all scans, pick one, and load with a click.
.zip ParaVision
